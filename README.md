# Movie_recommendation_system
A Machine Learning project that recommends movies to users based on their ratings and preferences using Collaborative Filtering. This system analyzes user behavior and identifies similar movies by calculating correlations between movie ratings.



🚀 Features
• Load and preprocess movie rating datasets
• Merge movie information with user ratings
• Analyze average movie ratings
• Visualize rating distributions
• Create a User-Movie Matrix
• Find similar movies using correlation
• Generate personalized movie recommendations
• Filter recommendations based on popularity

📈 Example

Input Movie
Star Wars (1977)

Recommended Movies
• Empire Strikes Back (1980)
• Return of the Jedi (1983)
• Raiders of the Lost Ark (1981)
• Terminator 2 (1991)
